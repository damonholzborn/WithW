Welcome to with w/. This guide is a cheat sheet for navigating the Mannequins w/ eurorack module. I hope that you’ll find this page a handy map to getting around your w/.

In addition to organzing the material into a (hopefully) easily digestible format, this guide is designed to be mobile-friendly, so you can view it without being tied to your computer. You can save the guide to the homescreen of your mobile device for easy mobile access.

If you find any errors or have any suggestions, please don’t hesitate to send them to damon@rustleworks.com. The code is open sourced under the MIT License. If you’d like to clone it for your own purposes, you can find the source at Github.

Happy Bleeping!

LIVE CV Control

Starting with firmware 1.2, THIS or THAT can be assigned alternate behaviors in LIVE mode. Enter the selector to change THIS/THAT modes by pressing record while inserting a cable into THIS or THAT, then select the new mode by pressing the button indicated below. To mix/match modes, press and hold your THIS selection then press your THAT selection.

dub

toggle recording and level (default)
select with play
THIS
input: trigger
toggle recording (punch in/out)
while playing
THAT
input: 0 to +5v
playback level (5v is quieter)
while recording
THAT
input: 0 to -5v
overdub (0v) to overwrite (-5v) — previous material not heard
input: 0 to +5v
overdub (0v) to overwrite (5v) — previous material is heard

transport

cues and speed (identical to NAV)
select with ▲
THIS
input: -5v to +5v
when playhead reaches a cue point, jumps to the cue indicated by THIS voltage — all cue points are evenly distributed
THAT
input: -5v to +5v
tape speed — fast backwards (-5v) to fast forwards (+5v)

signal

location ramp output and boundry trigger
select with loop
THIS
output: -5v to +5v
outputs voltage according to loop position (-5V start to +5v end)
THAT
output: trigger
sends a trigger whenever the play head hits a cue point

caption

record CV
select with record
THIS
input: -5v to +5v
CV to be recorded
THAT
output: -5v to +5v
recorded CV output

sample

jump cues and v/octave pitch shift (similar to CUE)
select with ▼
THIS
input: trigger
go to the most recently passed cue immediately — attaching a parallel static voltage (-1V to 1V) allows it to jump forward or backward by a single cue
THAT
input: -5v to +5v
v/octave tape speed

CUE CV Control

THIS
input: gate
play (and loop) while gate is high
THIS
input: -2v to + 2v
a ssmall CV in parallel with gate will perform a cue motion on next gate — -1v to -2v to select previous cue, +1 to +2 to select next cue
